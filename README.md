# ShopEase ‚Äî Ecommerce Frontend

A modern ecommerce storefront built with **React + Vite + Tailwind CSS**.

## Tech Stack
- React 18
- Vite
- Tailwind CSS
- React Router DOM

## Pages
- `/` ‚Äî Home page
- `/login` ‚Äî Customer login
- `/signup` ‚Äî Customer registration
- `/products` ‚Äî Product catalog with search and pagination
- `/cart` ‚Äî Shopping cart
- `/addresses` ‚Äî Shipping address management
- `/checkout` ‚Äî Order checkout
- `/orders` ‚Äî Order history
- `/orders/:id` ‚Äî Order detail
- `/account` ‚Äî User profile
- `/wishlist` ‚Äî Saved items

## Getting Started

```bash
npm install
npm run dev
```
